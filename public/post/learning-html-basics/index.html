Basic HTML

HTML is a tag-based language, it is built with elements. Elements have start and end tags, the end tag being identical except it starts with a forward slash. Elements can be within elements. Some tags like img or br are empty tags, there is only one (see below examples).

  • !DOCTYPE

  • Lists
  • Style

!DOCTYPE

The top line of the html file has this code

<!DOCTYPE html>

Signals that the file is written in HTML5 code. The entire body of the html document is surround by the tag html.

<!DOCTYPE html>
<html> 

</html>

Structure

The actual page is split into the head, the meta data of the page, and the body, the actual body of the page. Metadata is information about the page that isnt displayed.

<!DOCTYPE html>
<html>
   <head>
   
   </head>
   
   <body>
   
   </body>
</html>

Text & Images

Within the header you can set the title, the text displayed on the browser tab. Within the body, titles and subtitles can be made with h1 through h6, largest to smallest. Regular text can be written within paragraphs, p. A line break can be inserted anywhere with br. Images in the same folder as the html document can be inserted with img. A horizontal rule (horizontal line to speerate content) can be inserted with hr.

<!DOCTYPE html>
<html>
   <head>
      <title> Title </title>
   </head>
   
   <body>
      <h1> Biggest Header</h1>
      <h6>Smallest Header</6>
      <p> Regular paragraph text</p>
      <p> Broken <br> Paragraph</P>
      <hr>
      
      <img src="example.png" alt="example alt text" width="" height="">
   </body>
</html

Within the img tag, src signifies the image file name, alt represents the text that will show if for some reason the image cannot be displayed. Width and height set the width and height of the image.

Text within p tags cannot be formamted with spaces and line breaks without tags. Pre tags can be used to retain line breaks and extra spaces between words.

<!DOCTYPE html>
<html>
   <body>
      <pre>
      Line         breaks
      and     spaces    maintained
      </pre>
      
      <p>
      Line       breaks   
      and    spaces   NOT   maintained
      </p>
   </body>
</html>

Lists

You can make two types of simple lists, ordered and unordered. Unordered lists are marked with bullet points, ordered lists are marked with increasing numbers. ul and ol determine unordered/ordered, and every list item is denoted with li.

<!DOCTYPE html>
<html>
   <body>
   
      <ul>
         <li> Ex List Item </li>
      </ul>
      
      <ol>
         <li> Ex List Item </li>
      </ol>
      
   </body>
</html>

####Style Color, font, and size can be set with the style attribute. Attributes go within the start tag of an element, and effect only that element. Other attributes are img.

Style’s use CSS to change color, size, etc.

lang: Attribute for html to set language title: Attribute for p to create tooltip background-color: Attribute for body to set background color

Attribute text should be in double quotes, or single quotes if neccesary.

<html lang="en">
   <body>
      <p style="color: red"> Colored Text </p>
      <p title="Tooltip when hovered"> Paragraph </p>
   </body>
</html>
